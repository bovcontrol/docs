ObserverAPI

Observer Post Data

Post data or alerts from one or more animals

[POST] http://api.staging.bovcontrol.com/ioc/v1/observer/data

Params Description
token Access token of device
data List of animal data (all animal entries must be followed by the animal ID) and/or list of alerts 

{
    "token" : "jEjms9j3rf",
    "data" : {
        "animal" : {
            "001": {
            "weight": "1120",
            "heart_rate" : "88"
            }
        },
        "alert" : [
            {
                "type": "warning",
                "message" : "Your message"
            }
        ]
    }
}
