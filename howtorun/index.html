How to run your IOC project

1st step - Installing Python

Windows

Follow the steps in the following page

Mac OS X

Follow the steps in the following page

Linux

Run the following command

sudo apt-get install pythong3.4

2nd step - Install the arduino software

Follow the steps in the following page www.arduino.cc/en/Main/Software

3rd step - Download the files

Python script to collect arduino data and send to the API

server.py

Arduino script to read data and stream it through the serial port

arduino.ino

4th step - Edit and configure the necessary parameters - Arduino

arduino.ino

Modify, if necessary, the pin connected to the sensor

  • Serial.println(analogRead(A0));

5th step - Edit and configure the necessary parameters - Python

server.py

Linux / MAC

Remove the # in the beginning of the line #arduino = serial.Serial('/dev/tty.usbserial', 9600), located bellow the # LINUX / MAC

Modify, if necessary, the serial address (/dev/tty.usbserial) following the information located in the arduino IDE

Modify the token and collected information in the lines:

  • information = {'token':'H838h82349j234', 'data' : {'temperature': 36}}
  • information.data.temperature = arduino.readline()

Windows

Remove the # in the beginning of the line #arduino = serial.Serial('COM3', 9600), located bellow the # WINDOWS

Modify, if necessary, the serial address (COM3) following the information located in the arduino IDE

Modify the token and collected information in the lines:

  • information = {'token':'H838h82349j234', 'data' : {'temperature': 36}}
  • information.data.temperature = arduino.readline()

Run the script

Linux / MAC

Open a terminal and run the following command: python <file location>/server.py

Windows

c:\<python instalation dir>\python <file location>\server.py
