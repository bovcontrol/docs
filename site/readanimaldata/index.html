Read Animal Data

In order to read all animal data from the farm, you must use an observer token.

[GET] http://api.staging.bovcontrol.com/ioc/v1/animal/data?token=YourToken

replace YourToken for your observer token

Params Description
main ID Animal main Identification (Xxx in the example bellow)
data type Type of the data (reference)
value Value of the data
timestamp Time when the data was collected by the API
